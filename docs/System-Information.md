## Objective

Collect basic Linux system information.

## Commands

### Check hostname

```bash
hostname
```

### Current user

```bash
whoami
```

### Current directory

```bash
pwd
```

### Logged in users

```bash
who
```

### Operating System

```bash
cat /etc/os-release
```

### Kernel Version

```bash
uname -r
```

### Architecture

```bash
uname -m
```

### Uptime

```bash
uptime
```

### Date & Time

```bash
timedatectl
```

## Outcome

Successfully collected system information from Ubuntu Server.

# Enterprise Linux Server Administration & Automation Lab

## Project Overview

This project demonstrates enterprise-level Linux Server Administration using Ubuntu Server running on AWS EC2.

## Technologies

- AWS EC2
- Ubuntu Linux
- Bash Scripting
- Git
- Linux Administration
- SSH
- Networking
- Storage Management
- Firewall
- Monitoring

## Project Goals

- Learn Linux Administration
- Practice Enterprise Server Management
- Automate repetitive tasks using Bash
- Monitor server health
- Configure security
- Manage users and permissions
